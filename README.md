React frontend for the SDAT + DevOps Final Sprint project.  
This application displays airport arrivals and departures and provides an admin interface to manage flights.

---

## Tech Stack
- React (Vite)
- Ant Design
- Axios
- React Router
- Day.js

---

### Public Dashboard
- View airport arrivals and departures
- Select airport from dropdown
- Switch between Arrivals / Departures tabs
- Real-time data from backend API

### Admin Flights
- View all flights
- Create new flights
- Edit existing flights
- Delete flights
